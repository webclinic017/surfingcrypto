surfingcrypto.plotting.CoinFigure¶

class surfingcrypto.plotting.CoinFigure(kind='default', trendlines=False, graphstart='1-1-2021', *args, **kwargs)¶

Bases: surfingcrypto.ts.TS

This objects are complex matplotlib figures predisponed to fit OHLC + TA indicators data. Inherits from TS module, so it has attributes such CoinFigure.df to access the data in an easier way.

Note

There are two styles implemented at the moment: default that is candlestick + volume and ta that is candlecticks, volume + technical indicators.

Parameters

  • kind (str) – string representing desired style of plot.

  • trendlines (bool) – UNDER DEVELOPEMENT! - plot also trendlines calculated with src.trend_line class.

  • graphstart (str) – date string in d-m-Y format (or relative from today eg. 1 month: 1m,3 month: 3m) from which to start the graph.

  • **kwargs – Keyword arguments to TS module.
