surfingcrypto.config.config¶

class surfingcrypto.config.config(config_folder, data_folder=None)¶

Bases: object

Class for the package configuration. Contains API keys and user-specified parametrization of execution.

Note

data_folder is optional. If not specified checks if there is a data directory in the parent directory. If not, it will be created.

Parameters

  • config_folder (str) – ABSOLUTE path to config folder.

  • data_folder (str,optional) – ABSOLUTE path to data folder
