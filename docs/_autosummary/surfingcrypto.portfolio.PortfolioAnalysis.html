surfingcrypto.portfolio.PortfolioAnalysis¶

class surfingcrypto.portfolio.PortfolioAnalysis(mycoinbase_df)¶

Bases: object

This class is a Coinbase portfolio analysis.

Has methods for assessing portfolio values, grouping transactions by type, calculate total fees, etc.

With Portfolio.Analysis.standardize() it is possible to adapt the surfingcrypto.coinbase.MyCoinbase object to a standard form where there are only buy or sell trades and the amounts are always positive.

This standard portfolio allows to reconstruct easily the timeline of the portfolio value, at each given instant in time.
