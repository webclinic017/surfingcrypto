surfingcrypto.ts.TS¶

class surfingcrypto.ts.TS(configuration, coin=None)¶

Bases: object

This is an time-series oriented crypto price data object.

Note

Data can be downloaded calling `surfingcrypto.scraper.Scraper` object.

Parameters

  • configuration (surfingcrypto.config.config) – configuration object

  • coin (str) – string representing the crypto coin of choice, eg. BTC,ETH

df¶

dataframe with datetime index of ohlc data. Could store also TA indicators if these are computed invoking the relative method.

Type

pandas.DataFrame

ta_params¶

dictionary containing TA parametrization

Type

candlesticks(ax, volume=False, vol_ax=None, style='candlesticks')¶

plotting candlesticks into a matplotlib.axes.Axes object.

Parameters

  • ax (matplotlib.axes.Axes) – matplotlib ax to plot candlesticks into.

  • volume (bool,optional) – plot volume histogram data into a another ax specified with vol_ax

  • vol_ax (matplotlib.axes.Axes) – matplotlib ax to plot volume histogram into.

  • style (str,optional) – style of plotting candlesticks, candlesticks is default but ohlc style is used in BB bands plotting.
