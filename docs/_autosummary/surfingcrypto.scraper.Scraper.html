surfingcrypto.scraper.Scraper¶

class surfingcrypto.scraper.Scraper(configuration)¶

Bases: object

This is a wrapper of GitHub repo cryptocmd that scrapes data from coinmarketcap.com. It scrapes data for the crypto coins specified in the keys of the customizable coins.json file. Saves data in a *.csv file stored in data/ts/. Checks if the file exists, if not it is downloaded. If it exists, it check last element of datetime index and compares it with today’s date. If required, it updates the *.csv file to today’s date.
